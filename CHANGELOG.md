# 1.0.0 (2026-04-11)

Initial release. Forked from [ant-design/pro-components](https://github.com/ant-design/pro-components) and modernized with the following changes:

- Merged all packages into a single package `@xxlabs/pro-components`
- Upgraded to React 19
- Replaced build toolchain with tsdown
- Added ESLint (oxlint) and Prettier (oxfmt) for code quality
